2.JavaIntroList
　JupyterLabを用いた学習教材と進捗管理を行うもの。最初にJavaとついている
　ものが教材部分。checkerが進捗管理を行うファイルである。
　利用にはJupyterLabを用いることのできる環境(Anaconda推奨)とJavaカーネル
　のインストール、Modcellのインストールが必要。
　詳細は下記に記載

必要な環境
	Anaconda(JupyterLabが動けば別のものでもよいがサポート対象外である)
	最新のJavaとJavacの用意(Open JDK11)
	JupyterへのJavaカーネルのインストール(IJava)
	Modcellの導入

	インストールの参考サイト
	Anaconda公式サイト
	(https://www.anaconda.com/)
	JAVAーJupyter labでJavaを使えるようにする
	(https://blowup-bbs.com/jupyter-lab_java/)
	GitHub:SpencerPark
	(https://github.com/SpencerPark/IJava#installing)
	Qitta: Jupyter notebook で他の notebook の特定のセルをインポートする
	(https://qiita.com/wsuzume/items/77dce2d54dea9cb7751c)
